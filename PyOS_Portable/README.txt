PyOS Portable Shell
=================

This is a portable version of PyOS Shell that can be run on any Windows computer.

Instructions:
1. Copy the entire PyOS_Portable folder to the target computer
2. Run PyOSBoot.exe (requires administrator privileges)

Features:
- File management (create, delete, edit files)
- Directory navigation
- Application launching
- System commands
- Built-in text editor

Commands:
- ls: List files in current directory
- cd: Change directory
- clear: Clear screen
- open: Open and edit files
- app: Launch applications
- make_file: Create new files
- delete: Delete files
- exit: Exit the shell (with option to shutdown)
- reboot: Restart the system

Note: Some features require administrator privileges to work properly.
